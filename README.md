# ⛏️ MagicOre
Download the mod [HERE](https://github.com/Fazle-Rabbi-Rounok/magicOre/releases/download/mod/magicore-1.0.0.jar) <br>
This is my first minecraft mod<br>
Just made it for fun purpose to explore how these things are made<br>

A lightweight Fabric Minecraft mod that makes mining more exciting by giving **bonus magic loot** whenever you mine ores.

Instead of replacing vanilla drops, **MagicOre keeps the normal ore drops** and adds an extra randomly selected reward based on the ore you mined.

Perfect for playing with friends on a private server.

---

## ✨ Features

- 🎁 Bonus random loot from every vanilla ore
- ⛏️ Keeps original Minecraft ore drops
- 🎲 Weighted reward system
- 💎 Different reward pool for each ore type
- 🛡️ Rewards include:
  - Ores & Ingots
  - Weapons
  - Armor
  - Food
  - Utility Items
  - Rare Items
- ⭐ Extremely rare legendary rewards (Elytra, Netherite, etc.)
- 🌍 Multiplayer compatible
- ⚡ Lightweight and simple
- 🧩 Fabric Mod Loader

## Example

Mining a Diamond Ore might give:

- Vanilla Diamond
- Diamond Pickaxe
- Totem of Undying
- Netherite Scrap
- Beacon
- Elytra *(extremely rare)*

Every mined ore always gives its normal Minecraft drop plus one random bonus reward.

---

## Loot System

Each ore has its own weighted loot pool.

Common ores mainly give useful early-game resources.

Rare ores have a chance to give much stronger rewards.

The reward system is intentionally weighted so that legendary items remain very rare.

## Installation

### Requirements

- Minecraft 26.2
- Fabric Loader
- Fabric API

### Install

1. Install Fabric Loader.
2. Install Fabric API.
3. Place `magicore-x.x.x.jar` inside:

```
.minecraft/mods
```

For multiplayer:

- Install the mod on the server.
- Install the same version on every client.
